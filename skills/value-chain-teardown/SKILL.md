# 产业链全栈拆解 (Value-Chain Teardown)

把一个复杂赛道讲清楚、并落到"哪个环节最值得投"的可复用方法论。它的母题是一句话：

> **跟着钱（单位成本）一路往上游走，逐层问"这是什么生意、龙头是谁、现在贵不贵、未来往哪走"，
> 在最上游找真·咽喉，但只在"咽喉 + 好生意 + 不贵"三者同时成立处停下。**

本 skill 蒸馏自一次完整的「AI 数据中心全栈拆解」实战（见文末"参考实战"），但**方法论与赛道无关**，
可直接迁移到 EV/电池、机器人/具身智能、半导体设备、航空航天、国防、光伏、创新药 CXO 等任何
"硬件 / 供应链 / BOM 驱动"的赛道。

> **重要定位**：本 skill 产出的是**研究框架与尽调底稿**，**不构成投资建议**，使用者非持牌投资顾问。
> 强周期行业的基本面与股价可能短期剧烈背离；一切倍数/份额/占比中带"约/E/示意"的为方向性估算，不宜作精确引用。

---

## 第零步：数据要求（反幻觉硬约束 · 不可跳过）

在给出**任何**标的"贵不贵 / 值不值 / 谁赢"判断之前，必须先拉真实数据，否则只做定性、明确标注"待数据验证"：

1. **实时行情/估值** → 优先 IBKR MCP（`search_contracts` 解析 contract_id → `get_price_snapshot` 取 last/YTD/52周区间/`misc_statistics`；`get_account_positions` 看用户真实持仓）；或 Vibetrading-Michael MCP；或 Web 搜索一手来源（公司 8-K/财报、TrendForce、机构）。
2. **盈利质量**（毛利率 / 营业利润率 / 净利率 / ROIC / ROE / 定价权）→ 财报与一手披露，不靠记忆。
3. **份额/占比/市场规模** → 多源交叉，口径不一时取范围并注明。
4. **凡是推断出来的成本占比、热力图分值、未来比例** → 一律标注"方向性/示意"，并说明依据。
5. **不确定就承认**：宁可少说、标"待核实"，绝不编造归因或精确数字。

## 七步拆解流水线

### Step 1 · 锚定一个"单位成本拆解"作为主轴
找到该赛道一个**有代表性的终端单元**，把它的物料/成本结构按占比**从大到小**列出来——这是整份拆解的骨架。
- AI 数据中心 → 一台 Rubin NVL72 机柜（$7.8M：GPU 51% / 内存 26% / 网络·光·被动 21% / CPU 2%）。
- 迁移示例：EV → 一辆车的 BOM（电池包 / 电驱 / 电控 / 智驾域控 / 车身）；机器人 → 一台人形机器人（关节执行器 / 减速器 / 传感 / 算力 / 电池）；创新药 → 一条新药管线的成本/价值链（靶点发现 / CRO / CDMO / 临床 / 商业化）。
- **要点**：用真实成本占比，注明来源（如摩根士丹利 teardown）；占比即"钱在哪"，后续每一层都锚定它。

### Step 2 · 逐层下钻（统一模板）
按成本占比从大到小，对每一层用**同一张模板**回答四问，保证结构整齐、可比：
1. **是什么产业**（这一层在干什么、为什么重要）
2. **龙头是谁**（竞争格局、份额、梯队）
3. **现在贵不贵**（远期 PE / EV-EBITDA / PB，配盈利质量；用第零步的实时数据）
4. **未来往哪走**（周期位置、技术路线、供需、颠覆风险）

### Step 3 · 钻到"Tier 0 地基"（上游封装/材料/设备/IP）
终端单元拆完，继续往上游钻一层——决定良率与产能、却不在 BOM 显眼处的**封装 / 材料 / 设备 / 关键 IP**。
这里往往藏着**真·咽喉**（如 AI 链的 CoWoS 封装、味之素 ABF 膜、日本 EUV 光刻胶）。规律：越上游越集中、越隐蔽、越常被错误分类。

### Step 4 · "咽喉 ≠ 机会"纪律（核心筛选器）
**是不是咽喉，和它是不是好投资，是两回事。** 每个候选环节必须同时过两关，做成"盈利质量 × 估值"2×2：
- **关一：咽喉能不能转成真金白银** —— 毛利率 / 营业利润率 / **ROIC**（低于资本成本=毁灭价值）/ 定价权（能否提价）。
- **关二：现在贵不贵** —— 估值倍数 + 距历史区间/52周高点位置 + "涨幅是否已透支景气"。
- **四象限**：① 甜区＝好经济+便宜（极少，最该重点尽调）；② 好经济+但贵（设备/已重估龙头）；③ 好经济+被综合体稀释（错误分类 mispricing，最肥猎物，见棱镜⑥）；④ 雷区＝差经济+贵（人人皆知的卡点常落此处，如低 ROIC 的 OSAT 被炒到高 PE）。
- 反例教训：卡点 ≠ 机会。最显眼的瓶颈往往回报最差、定价最满。

### Step 5 · 套用"多棱镜洞察"（见下节清单，逐条过）
不要只做线性拆解，用一组横切棱镜重新审视全栈，挖出非显然的结论。

### Step 6 · 收口与落点
把全部下钻收敛成**可执行的判断**：
- 哪个环节"最便宜 + 最核心"？（常出现估值倒挂：最该买的反而最便宜）
- 哪些是"路线无关 + 需求无关"的双重免疫赢家？（确定性最高）
- 每个候选的**否决项**（单点风险/管制/替代技术/估值透支/盈利在周期顶）。
- 一句话主结论 + 唯一的近端证伪变量（如"AI capex 会不会踩刹车"）。

### Step 7 · 品牌化交付（见"交付规范"）
产出至行·Zenith 品牌的 PDF（WeasyPrint 或 playwright print）或可滚动 HTML，逻辑＝成本主轴 → 逐层下钻 → Tier 0 → 多棱镜洞察 → 全栈总览表 → 免责。

## 多棱镜洞察清单（可复用，逐条自检是否适用本赛道）

| # | 棱镜 | 问的问题 | AI 链中的实例 |
|---|---|---|---|
| ① | **外溢级联** | 一个环节的瓶颈/涨价会沿链条传导到哪些下游？是否形成自我强化循环？ | HBM 产能挤占 → DRAM → NAND → HDD 全栈售罄 |
| ② | **颠覆敏感度热力图** | 每个龙头对某外生冲击（国产替代/技术替代/政策）的暴露度打分（敞口 × 可竞争性）。 | 中国冲击：纯 NAND 9.0 最脆，HBM 龙头 4.6 最免疫 |
| ③ | **估值倒挂** | 是不是"越靠近皇冠/质量越高的反而越便宜"？为什么？（峰值盈利低 PE vs 结构性溢价高 PE） | HBM 龙头 ~6x 却最强；HDD/CPU 反而更贵 |
| ④ | **路线无关** | 终端有多条竞争路线（如商用 GPU vs 自研 ASIC）时，哪些环节"押谁都赢"？ | HBM、台积电 CoWoS 通吃，是"瑞士" |
| ⑤ | **需求/负载分层** | 同一条栈在不同需求/负载下成本重心如何迁移？主轴单元代表的是哪一种？ | 训练→推理→agentic，CPU 占比 2%→16%，GPU 收缩 |
| ⑥ | **错误分类 / mispricing** | 最好的垄断经济学是不是被埋在综合体/错误的分析师覆盖里？ | 味之素（ABF 50% 利润率藏在食品财报）、信越（硅片藏在化工） |
| ⑦ | **滞后兑现 vs 结构性溢价** | 高 PE 里多少是"涨价/盈利还没兑现的滞后"（会压缩），多少是真实结构性溢价（不会消失）？ | HDD：PE 从 137x 压到 34x 是滞后，~30–50x 是地板 |
| ⑧ | **盯真实持仓** | 把结论落到用户 IBKR 的真实仓位上，区分"估值便宜 ≠ 价格没涨"。 | 海力士/铠侠/美光已翻数倍贴近高点，"便宜"是峰值盈利上的便宜 |

## 交付规范（至行 · Zenith Research 视觉系统）

- **品牌主题**：深海军蓝 `#0B1C33 / #16324F` + 金色 `#C9A227 / #E3C766`，米白纸面 `#FAF8F3`；中文正文，CJK 字体。
- **数据用等宽字体**（mono），做"研究终端"质感；所有倍数/百分比 tabular-nums 对齐。
- **结构（PDF 或滚动 HTML 一致）**：封面/Hero（主结论） → 成本主轴彩条（签名元素，复用为各层定位条） → 逐层下钻（统一四问模板 + 估值表） → Tier 0 地基（三大咽喉 + 盈利×估值表 + ROIC×PE 散点图 + 上游垄断条） → 多棱镜洞察（外溢级联流程图 / 敏感度热力图 / 估值倒挂表） → 全栈总览表（占比·龙头·远期PE·敞口·性价比·未来） → 免责声明。
- **图表**：matplotlib（品牌色）或原生 HTML/CSS（成本条/热力图网格/内联 SVG 散点）。HTML 版用 IntersectionObserver 做 scroll-reveal，但**导出 PDF 前必须强制 `.reveal{opacity:1}` 并移除吸顶导航**，再用 playwright `pdf(print_background=True, displayHeaderFooter=True)` 出 A4。
- **每张推断图必带脚注**："占比/分值为方向性示意"。
- **每份必带免责**："研究参考、非投资建议、强周期、以一手数据为准"。

## 跨赛道复用映射（这套打法怎么搬到别的赛道）

把每个赛道翻译成本 skill 的四个"槽位"，方法论原样套用：

| 槽位 | AI 数据中心 | EV / 电池 | 机器人 / 具身智能 | 创新药 CXO |
|---|---|---|---|---|
| **主轴单元(Step1)** | Rubin NVL72 机柜 BOM | 一辆 EV 的 BOM | 一台人形机器人 BOM | 一条管线的价值链 |
| **最大成本头** | GPU/加速器 | 电池包 | 关节执行器/减速器 | 临床/商业化 |
| **Tier 0 地基** | CoWoS/ABF/EUV胶 | 正负极材料/隔膜/电解液/锂钴镍 | 谐波减速器/丝杠/力矩传感/高端轴承 | 反应试剂/酶/一次性耗材/专利分子 |
| **真·咽喉典型** | 味之素 ABF、台积电 CoWoS | 隔膜涂覆、上游锂资源、设备 | 谐波减速器、空心杯电机、6 维力传感 | 高端 CDMO 产能、关键专利 |

通用迁移口诀：**先找"那台机柜"（主轴单元）→ 跟着成本占比往上游钻 → 在最上游找咽喉 → 用"盈利×估值"筛掉伪机会 → 用多棱镜挖非显然结论 → 落到真实持仓**。

## 参考实战

「AI 数据中心全栈拆解」：从 Rubin 机柜 $7.8M 成本 → ①GPU 51% / ②内存 26% / ③光·被动 21% / ④HDD / ⑤CPU 2%
→ Tier 0（CoWoS / ABF 味之素 / EUV 光刻胶；Amkor 雷区 vs 信越甜区）→ ⑥多条路线（GPU vs ASIC，HBM 是瑞士）
→ 负载分层（训练/推理/agentic 成本重心迁移）→ 三大洞察（外溢级联 / 中国敏感度热力图 / 估值倒挂）→ 全栈总览表。
主结论：最便宜+最核心+路线无关+负载无关的是 HBM/DRAM 龙头；唯一近端证伪变量＝AI capex 是否踩刹车。
