## 案例 1：2021-04-09 东方财富 002047 反包板（失败 + 反思）

| 项 | 内容 |
|---|---|
| 标的 | 东方财富 002047 |
| 日期 | 2021-04-09 |
| 操作类型 | **打反包板 + 后面做了两天 T** |
| 节奏 | 人气老妖股的二波打反包板 |
| 结果 | **两个 T 从盈亏截图看是失败的，最后大面走** |
| 光神原话 | "**还不如拿着不动**，不 T 最后跌回成本位可能不亏，或小亏" |
| **学习要点** | **持仓时少做 T**——龙头/人气股持仓时频繁做 T 反而降低胜率。这是光神公开承认的常见失误模式 |

## 案例 4：2021-04-14 人气庄股高位首次大分歧后上车（次日大面）

| 项 | 内容 |
|---|---|
| 标的 | 人气庄股 |
| 日期 | 2021-04-14 |
| 操作类型 | **高位首次大分歧后上车** |
| 结果 | **当天亏 2 个多点** |
| 光神原话 | "**这种看不懂，次日能这么强，感觉和碳中和龙头跌也有很大关系。这个市场很大一部分龙头资金就是围绕这些高标互相切换**" |
| 重要反思 | "**至少人工智能这一波和注册制新规后主流资金以中军做趋势为主**" |
| **学习要点** | **庄股的高位分歧上车风险极大**——即使首次大分歧也可能直接走弱。光神反思**注册制后主流资金从"高标互切"变成"中军做趋势"** |

## 案例 5：图解 1 总结（10 案例胜率）

| 项 | 内容 |
|---|---|
| 时间 | 2021-04 上半月 |
| 10 张图解结果 | **吃面 3 次（7 胜 3 负）** |
| 光神原话 | "**主流为主，分仓**，毕竟很多介入点都是高位，**全仓如果崩一次，面太大**" |
| 关键洞察 | "**人气主流高位打板较多，当下可能不实用了**。但**变化的只是表面，本质没有改变，就是紧跟主流资金，无限接近主流资金**。只是当时的主流资金是连扳人气高标的玩法，**现在变成了核心趋势中军的玩法**" |
| **学习要点** | **方法论的迁移**——光神早期靠"连板人气高标"，现在要换成"核心趋势中军"。但**核心永远是紧跟主流资金**，不变 |

## 案例 8：2021-06-18 三峡能源 600905 冲板就走（卖飞反思 - 经典）

| 项 | 内容 |
|---|---|
| 标的 | 三峡能源 600905 |
| 日期 | 2021-06-18 |
| 操作类型 | **人气次新牛股反包加强再次上车，并没有完全打板，可惜次日卖飞** |
| 失败描述 | "**第二天冲板就走，这种错误我也经常犯**" |
| 光神原话 | "**对于很多大成交的票，冲板在板上高成交，你会认为还会再炸开，决定先走再看，结果就卖飞**" |
| 重要反思 | "**当时碳中和的炒作此时已经太久，三峡能这么强也是我没想到的**" |
| **学习要点** | **大成交的票冲板时不要走**——按"还会再炸开"的心理卖出 = 大概率卖飞。这是光神公开承认的常见失误模式。**教训：拿住核心龙头的关键是抗住"以为要炸"的心理诱惑** |

## 怎样使用本案例库（给 Claude 自己）

B 类复盘任务流程：
1. **提取用户操作的核心要素**：标的、连板数、入场动作、持仓时长、退出方式
2. **判断是不是"核心人气龙头"** — 如果不是，对照杂毛失败案例
3. **匹配 1-2 个最接近案例**
4. **回 `playbook.md` 找光神会怎么处理**
5. **输出时引用案例号**

**重要原则**：光神的胜率约 70%，**盈亏比是关键**。复盘时不要因为一笔吃面就否定方法论——要看是不是符合光神 4 个关键反思（不 T / 不冲板就走 / 杂毛及时离场 / 高位分歧别上车）。

**注释**：第二组（莱茵生物 7 天）原书是另一位游资的案例不属于光神。这里整理的所有案例均出自一瞬流光本人交割单。
