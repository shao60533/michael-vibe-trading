## 案例 1：浙江建投第一波（主升浪锁仓 + 尾盘砸出）

| 项 | 内容 |
|---|---|
| 标的 | 浙江建投 002761 |
| 日期 | 2022-02-21 至 2022-03-14 |
| 主线 | 基建（合力强、核心逻辑硬） |
| 龙头地位 | 第一龙头 |
| 节奏 | 主升期 → 高潮期 |
| 操作 | 一路锁仓持有 |
| 卖出 | **2022-02-21 11:26 集合竞价卖出 3298w**（首波高点） |
| 卖出 | **2022-03-14 14:59 尾盘集合竞价**卖出 49.56 万股 + 32.73 万股 |
| 卖出金额 | 总卖出 144 万股，金额 3061w + 5000w |
| 陈小群原话 | "**不知道为什么就要抢就要抢，等了九天，最后一天就不能等，尾盘空了两万手**" |
| **学习要点** | **主升浪锁仓的极致演示**——9 天不动，最后一天尾盘集合竞价砸出。**只在尾盘卖**（多个时间点都在 11:26、14:59 这种关键时刻）。**不做差价**：盘中波动不动，全部集中尾盘 |

## 案例 3：中交地产（核按钮反核 - 模式 2 经典）

| 项 | 内容 |
|---|---|
| 标的 | 中交地产 000736 |
| 日期 | 2022-03-29（首次买入）至 2022-04-13（反核） |
| 主线 | 地产 |
| 节奏 | 主升期 → 高潮期 → **核按钮**反核 |
| 第一段操作 | 2022-03-29 买入（3 连板时介入），4-12 高点卖出 8530w |
| 第二段操作 | **2022-04-13 开盘被核按钮，反核买入 8604w** |
| 退出 | 持有 3 天后抛出，**大赚近 40%** |
| 陈小群手法 | 21 天 11 板大赚好几千万 |
| **学习要点** | **核按钮反核的教科书**——必须满足：①真正的第一龙头 ②核心逻辑过硬（地产主线没死） ③情绪还在主升期。反核条件齐备时，**核按钮反而是最便宜的入场点**。⚠️ 但这要求对情绪节点和龙头地位有极强把握，新手不要学 |

## 案例 4：特力 A（核按钮反核 + 复制中交地产）

| 项 | 内容 |
|---|---|
| 标的 | 特力 A 000025 |
| 日期 | 2022 年 5 月 -2022-06-07 |
| 主线 | 汽车补涨龙 |
| 第一段 | 两次买入 1573w + 1328w，最高点卖出 4311w |
| 第二段 | **2022-06-07 被核按钮，反核买入 3130w** |
| 退出 | 持有 2 天后卖出 4181w，**两天大赚 1000w**（次日地天板） |
| 作者评价 | "**特力 A 完美的复制了中交地产，就连操作都如此相似**，最高点卖出，核按钮接回来，每一次买卖都精准无误" |
| **学习要点** | **反核手法的可复制性**——陈小群对同类型题材（地产→汽车）用相同手法。**这不是运气，是对核按钮 + 主线情绪的系统化判断**。如果第一次反核成功，且环境类似，可以复用 |

## 案例 5：中通客车（主升浪后期介入 + 监管应对）

| 项 | 内容 |
|---|---|
| 标的 | 中通客车 000957 |
| 日期 | 2022-06-16 至 2022-07-06 |
| 主线 | 2022 年人气最高的妖股 |
| 介入时机 | **2022-06-16 分歧时低吸 3424w**（介入较晚） |
| 中间操作 | 6-20 做 T，6-21 三日榜共买入 10334w，卖出 9897w |
| 监管事件 | 遭遇监管停牌半个月 |
| 退出 | **2022-07-06 跌停开盘，全部抛出 4461w** |
| **学习要点** | **遇到监管停牌时的处理**——不犹豫，跌停开盘全部抛出。**风控大于一切**。这案例说明陈小群虽然格局强，但**遇到不可控因素（监管）也会果断止损** |

## 案例 6：剑桥科技（趋势龙头 30 天 200% 异动消化期 - 模式 5）

| 项 | 内容 |
|---|---|
| 标的 | 剑桥科技 603083 |
| 日期 | 2023-04-18 |
| 主线 | 光模块（AI 算力分支） |
| 介入逻辑 | "**起码有 30 天 200% 异动时间，越来越少**" → "**起码有 10% 空间**" |
| 入场 | 2023-04-18 周二买入 1.17 万手，均价 47.62 元，**合计 5579w** |
| 持仓 | 周三涨停锁仓，周四涨停做 T，周五卖出 1.16 万手均价 58.13 元，6786w |
| 利润 | **3 天大赚 1200w** |
| 陈小群原话 | "**24.25 号我都在买剑桥，是因为这个位置买，起码有 30 天 200% 异动时间，越来越少，这个位置买、起码有 10% 空间。市场总是不负我，25 号上了龙虎榜，26 号就涨停了**" |
| **学习要点** | **趋势龙头的入场标准**——异动监管披露（30 天 200%）后的整理期，是机构资金重新建仓的最好时机。**异动越来越少**意味着筹码集中度提升。这是一个**纯逻辑驱动**的入场判断，不靠任何技术形态 |

## 案例 7：新易盛（龙头切换 - 模式 3）

| 项 | 内容 |
|---|---|
| 标的 | 新易盛 300502 |
| 日期 | 2023-04-19（首次买入新易盛） |
| 主线 | 光模块（剑桥科技已经走趋势牛涨近两倍后） |
| 介入逻辑 | "**这是群神对新易盛的操作，当时龙头剑桥科技已经走趋势牛涨了近两倍，再次转强。群神直接上最强后排，这波拉升后排龙涨幅超过了龙头**" |
| **学习要点** | **龙头转强时上最强后排**——这不是常规"做后排"，而是**主线龙头确认走 N 波后，最强后排往往超龙头**。条件：①主线还在 ②龙头连续转强（不是首次启动）③后排是"最强后排"（不是杂毛） |

## 案例 8：城邦股份（卡龙头首次短板补涨 - 模式 4 / 与北京炒家对比）

| 项 | 内容 |
|---|---|
| 标的 | 城邦股份 600728 |
| 日期 | 2022-02-15 |
| 主线 | 基建（浙江建投是龙头） |
| 陈小群入场 | 2022-02-15 买入城邦股份 → **超级大肉** |
| 北京炒家对比 | **2022-02-14 打板进场城邦股份 → 次日割肉离场** |
| 作者评价 | "**此案例是陈小群对当时基建的后排做补涨，又是卡在龙头首次短板时间点，超级大肉。北京炒家当时是纯首板模式次日离场没毛病**。这就是**模式派和市场理解力派的不同玩法**" |
| **学习要点** | **理解力派 vs 模式派的最直观对比**——同一只票，差一天进场，结果完全不同。陈小群的"卡龙头首次短板时间点"需要：①对主线龙头节奏的精准判断 ②知道哪只是真正的卡位补涨龙（不是杂毛跟风）③敢在大家恐慌时进场 |

## 案例 9：万方发展（庄股摘帽 3-6 板 - 模式 6）

| 项 | 内容 |
|---|---|
| 标的 | 万方发展 000638 |
| 日期 | 2023-01-03 |
| 主线 | 庄股摘帽连板 |
| 性质 | "**当时的庄票，摘帽，庄家利用这摘帽直接拉连板打出高度，让做高标的资金去接盘**" |
| 陈小群操作 | 在 3-6 板之间介入（具体板位不详） |
| 退出标准 | "**最好还是只在 3 板到 6 板之间，相对安全一点**" |
| 反思 | "**目前这种玩法已经变了，至于后期市场还会不会重新回归以前的玩法，还不得而知**" |
| **学习要点** | **庄股摘帽是高风险玩法**——只在 3-6 板的安全区间。**关键反思**：注册制后这种玩法已经变了，2023 年之后的环境**不要套用**。如果在 2025 年还想做庄股，要降低预期、缩小仓位 |

## 案例 10：西安饮食（明牌强逻辑高换手反复做 - 模式 7）

| 项 | 内容 |
|---|---|
| 标的 | 西安饮食 000721 |
| 日期 | 2023-01-18 |
| 主线 | 消费放开第二波 |
| 性质 | "**西安饮食当时也是主力潜伏很早的庄票，尤其是第一波，主力老早就在运作这个放开预期，但是第二波这个放开炒消费的牌是非常明的**" |

## 怎样使用本案例库（给 Claude 自己）

B 类复盘任务流程：
1. **提取用户操作的核心要素**：标的、入场板位、持仓时长、是否第一龙头、是否核按钮
2. **按速查表匹配 1-2 个历史案例**
3. **回 `playbook.md` 找陈小群会怎么处理这种场景**
4. **输出时引用案例号**（如"这跟案例 3 中交地产的反核很像，但缺了情绪节点判断"）
5. **如果用户的操作不符合陈小群的任何模式，明确说明**——可能是误把陈小群手法套用在不适合的场景

**重要原则**：陈小群的手法**不可机械复制**——他的"龙头信仰锁仓"和"核按钮反核"都需要极强的情绪判断和资金量支持。在复盘时，**对成功要警惕是否运气好，对失败要看是不是缺少了某个前提条件**。
