## 案例 2：强势形态低吸（图解 6 案例 2 - 涅盘最爱模式）

| 项 | 内容 |
|---|---|
| 日期 | 2019-06 |
| 操作 | **低吸 + 突破加强势组合形态** |
| 结果 | **大肉** |
| 涅盘原话 | "**一个低吸大肉，突破加强势组合形态，这也是我最喜欢的低吸模式。但是涅盘也是小卖飞**" |
| **学习要点** | **强势形态低吸 = 突破 + 加强势 + 组合形态**——三要素叠加。这是涅盘最高确定性的低吸模式。**注意**：即使是涅盘最爱模式，也常常卖飞——**低吸成功后要敢拿** |

## 案例 4：尾盘竞价上金融（图解 1 案例 10 - 失败 + 反思）

| 项 | 内容 |
|---|---|
| 日期 | 2019-04 |
| 操作 | **尾盘竞价上金融** |
| 判断 | "当时指数的火热，认为金融可能要爆发" |
| 结果 | **次日金融高开但低杀，走在最低点，1 个点小面** |
| **学习要点** | **尾盘竞价上 = 明日预判驱动**——预判错就是吃面。涅盘的成功率高（吃面只是 1 个点小面），关键是**仓位轻 + 次日快速止损** |

## 案例 5：图解 1 总结（10 案例胜率）

| 项 | 内容 |
|---|---|
| 时段 | 2019-04 上半月 |
| 10 张图解结果 | **成功率比较高，吃面几次都是一两个点的小面，大肉不少** |
| 涅盘原话 | "操作中**有打板、有半路、有低吸、有尾盘竞价上**。所有操作都来自于**对当天盘面强势方向的跟随预判和对明日的预判**——明日不看好，一字板也走；明日看好，尾盘竞价也要上点仓位。**跟随强势方向和预判明日走势做的非常好**" |
| **学习要点** | 涅盘的胜率本质：**手法不重要，重要的是"盘面预判 + 明日预判"**。手法是分支，预判能力是树干 |

## 案例 6：打汽车板块龙头 4 板（图解 2 案例 3 - 大亏 + 反思）

| 项 | 内容 |
|---|---|
| 日期 | 2019-04 |
| 操作 | **打当时汽车板块龙头 4 板** |
| 结果 | **大亏** |
| 涅盘反思 | "**当天总结超级有问题，低级失误。如果看好这个板块，潜伏小弟就好**，第二天如果龙头继续涨停，小弟也能起来。即便今天避空，龙头明天还是有分歧跳水风险，也就 10 个点的收益，风险却巨大。**小弟没风险，收益却很大，小弟收益风险比比龙头高很多**" |
| **学习要点** | **看好板块时不要追高位龙头打板** —— 应该**潜伏低位小弟**。小弟的收益风险比远好于龙头打板。**这与一瞬流光"打高位龙头"完全相反**——涅盘更偏好用风控视角思考 |

| 项 | 内容 |
|---|---|
| 日期 | 2019-06 |
| 操作 | **潜伏当时汽车零部件概念** |

## 案例 8：图解 5 大跌环境最大单笔失误（涅盘最严重的一次）

| 项 | 内容 |
|---|---|
| 日期 | 2019-04 末至 5 月初 |
| 大盘环境 | 即将大跌调整 |
| 操作问题 | **涅盘在大跌环境下出手，回撤很大。从最高 2600 多回撤到 2100 多。回撤了近 500 个点。这是他在之前 300 个徘徊期过后最大的一次回撤** |
| 涅盘反思 | "上期图解指数在顶部感觉要走弱时，涅盘是感受到了的，虽有试探性操作，但总体仓位不重。**但是在上期我们已经感受到涅盘的成功率不高了。这就是我之前说过的一点，当以前这种位置你进去都能吃肉，现在总是吃面时，这个时候就要小心**" |
| 二次错误 | "**指数才跌几天，这次涅盘操作的仓位上就开始加大，达到七八成。在补跌情况下连吃跌停**" |
| 后续 | "**之后并没有马上停止，继续维持着较高的仓位操作，手法上低吸多了，但是部分低吸明显的中高位，补跌下继续吃面**" |
| 作者比较 | "**总体来说可能和前段时间北京炒家超跌低吸，想一把捞回来有点像。脱离了自己最擅长的模式，低估了指数的杀跌风险**" |
| **学习要点** | **大跌时不要硬上**——涅盘的最大教训。**三个关键警告**：①当成功率开始下降时要警觉 ②大跌后不要急着加仓 ③不要脱离最擅长的模式硬上。**这与北京炒家 8 个月血亏的本质相同**——脱离纪律 = 灾难 |

## 案例 9：图解 8 案例 10 - 主观 YY（"前 8 张白干"）

| 项 | 内容 |
|---|---|
| 日期 | 2019-06-04 |
| 标的 | 迪普科技（次新二板） |
| 操作 | **打次新二板** |
| 结果 | "**卖的很差，次日高开快速下杀没有离场，后面封死跌停**" |
| 涅盘 YY 理由 | "**涅盘认为他是次新卡前排打头的，应该有资金去做回流**" |
| 作者评价 | "**这个 YY 我的觉得是不够客观的，因为前一天图 9，次新的这种首板都炸成这死样，肯定要走的。何况是一个攻击前高的位置。我认为这种属于涅盘的低级失误，涅盘之前后排不给力，二板一字都卖过两回的人，不应该犯这个低级错误。但是一旦 YY 起来，你会犯下之后你自己都觉得可笑的错误**" |
| 涅盘原话 | "**一直保持相对客观是比较难的一件事情**" |
| 当日盈亏 | -1,331,324.92 (-5.59%) |
| 影响 | **"涅盘前 8 张都做得非常好，但是后两张的失误几乎让前 8 张白干"** |
| **学习要点** | **YY 是最大的敌人**——即使顶级游资也会犯。**警告信号**：当你开始"找理由证明持仓正确"而不是"客观评估盘面"时，就是 YY 开始。**预防方法**：每笔操作前自问"如果今天是第一次看到这只票，我会买吗？" |

## 案例 10：低吸银行避空（防守经典）

| 项 | 内容 |
|---|---|
| 日期 | 2019-04-17 |
| 操作 | **感受到有风险，低吸银行防守** |
| 涅盘评价 | "**高手对盘面的感知就是强**" |
| 后续 | "**确实到顶，之后几天就开始大跌。涅盘当天重仓低吸了 4 只银行，都差不多**" |
| 实际效果 | "**银行并没有护盘，次日都是小小面离场**" |
| 涅盘反思 | "**所以我之前强调，判断指数要不要调整，从盘面情绪上就能感受到**" |
| **学习要点** | **大跌前低吸银行 ≠ 一定赚钱**——银行未必护盘。但**仓位轻 + 快速止损**让损失可控（小小面）。**关键能力**：从盘面情绪判断指数调整——涅盘的核心能力之一 |

## 怎样使用本案例库（给 Claude 自己）

B 类复盘任务流程：
1. **提取用户操作的核心要素**：手法、当时盘面、是否主流板块、是否强势形态
2. **匹配 1-2 个最接近案例**
3. **判断是否符合涅盘的核心原则**：
   - ✅ 主流资金所在板块
   - ✅ 涨停强势组合
   - ✅ 强势形态低吸（最爱）
   - ✅ 明日预判驱动
4. **特别警惕涅盘公开反思的 3 大失误**：
   - 打高位龙头（应潜伏小弟）
   - 大跌时硬上（应空仓）
   - 主观 YY 找理由（应保持客观）
5. **回 `playbook.md` 找涅盘会怎么处理**

**重要原则**：涅盘是**全能型选手**，复盘时要看用户**是否在涅盘擅长的范围内**。**强势形态低吸是涅盘的金字招牌**——如果用户做的是这类操作，按高确定性评估；如果是高位追龙头、大跌抄底等，按涅盘明确反思的失误评估。
