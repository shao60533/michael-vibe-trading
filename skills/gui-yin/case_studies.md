| 逻辑层次 | **次优 — 博弹性** |

## 案例 6：2025-02-11 华丰科技（弱逻辑 - 反面）

| 项 | 内容 |
|---|---|
| 日期 | 2025-02-11 |
| 操作 | **低吸** |
| 标的 | 华丰科技 688629 |
| 性质 | 生产连接器的，偏硬件，擦边数据中心概念 |
| 类比 | 火锅店的那口锅上有棵螺丝是他做的 |
| 作者评价 | "**我感觉逻辑稍弱，盘中资金强度也不够**" |
| **学习要点** | **反面教材：逻辑稍弱时不要勉强**——归因自己也承认这单逻辑不够硬。**"火锅店螺丝"的比喻**——和主线的关联太弱（DeepSeek → 数据中心 → 服务器 → 连接器 → 螺丝），每跨一层关联度都会下降。**归因的反思**：即使是顶级游资也会偶尔做弱逻辑的票（"防止完全踏空"），但要承认这是次品而不是上品 |

## 案例 7：2025-02-11 上工申贝（次优 - 套利）

| 项 | 内容 |
|---|---|
| 日期 | 2025-02-11 |
| 操作 | **扫板 + 次日炸板平均 3 个多点离场** |
| 标的 | 上工申贝 600843 |
| 题材 | 低空经济（新题材） |
| 归因判断 | "**到了主线这个阶段，因为很多 20 厘米龙头都开始开板了，主线分歧加大，市场开始有人去博弈挖掘新题材**" |
| 归因坚持 | "**主线龙也不过首阴，主线结束还早，当时我的观点是要开始重视 10 厘米**" |
| 核心规则 | "**主线没挂，套利题材一般走不出高度**" |
| **学习要点** | **套利题材在主线未结束时走不出高度**——归因虽然去了上工申贝（套利低空经济），但**炸板后果断离场只赚 3 个点**。**关键洞察**：主线分歧期市场资金会去博弈新题材，但**主线没死的话新题材高度有限**——参与可以，**不要重仓**，赚一点就走 |

## 章末总结（归因原话）

> **当主线很强的时候，逻辑硬，怎么上就行，主要就是敢干。**
>
> **超级主线前期，基本都是卖飞多，踏空多。**
>
> **归因去的一些标的，挖掘的一些逻辑，有些还是很硬，个别不算特别硬。**
>
> **但有时我们挖掘也会发现，逻辑很硬发酵最猛的，开盘就是一字，当下自媒体时代催化太快了。**
>
> **很多时候不是我们不想去逻辑最硬的，而是不给机会。**
>
> **这个时候逻辑不是特别硬的也会去一点，防止完全踏空，有时我也经常这样。**

**关键洞察**：
1. **主线强时敢干** —— 不要纠结买点
2. **卖飞多踏空多** 是常态 —— 接受
3. **逻辑最硬的常常买不到** —— 一字板抓不到正常
4. **逻辑稍弱的也会去** —— 防止完全踏空，但要承认是次品

## 怎样使用本案例库（给 Claude 自己）

B 类复盘任务流程：
1. **提取用户操作的核心要素**：是否在风口？逻辑硬度如何？主线行情阶段？是否符合低吸 5 条件？
2. **匹配 1-2 个最接近案例**
3. **判断是否符合归因的核心原则**：
   - ✅ 在当下风口
   - ✅ 逻辑硬（基本面 / 事件 / 产业链）
   - ✅ 阴线出现在启动后
   - ✅ 板块仍在前期
   - ✅ 阴线杀跌时进场
4. **回 `playbook.md` 找归因会怎么处理**
5. **特别提醒**：
   - 如果用户在做"火锅店螺丝"级别的弱关联 → 提醒案例 6 反面教材
   - 如果用户在主线未死时博套利 → 提醒案例 7 走不出高度
   - 如果用户卖飞了 → 提醒案例 8 卖飞是常态
   - 如果用户加杠杆 → 提醒归因虽然用两融但是经验老到的产物

**重要原则**：归因是**逻辑驱动**的代表——复盘时**不要只看 K 线**，要还原**用户对逻辑的理解程度**。如果用户连标的逻辑都说不清楚，即使赚了也是运气；如果用户对逻辑理解透彻但操作失误，是技术问题，可以改善。
