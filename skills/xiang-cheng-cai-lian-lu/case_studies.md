## 案例 1：安里时代（人气股震荡后二次启动首板 - 最经典）

| 项 | 内容 |
|---|---|
| 标的 | 安里时代 600051 |
| 日期 | 2019-09-09 |
| 操作 | **加仓打首板** |
| 形态 | **人气股震荡后的二次启动打首板** |
| 同行评价 | "也是**最近最火游资一瞬流光比较喜欢的点**" |
| **学习要点** | **二次启动首板 = 一次启动确认题材有热度 + 整理消化筹码**。这是连接采莲路（控回撤派）和一瞬流光（高位接力派）的共同心头好——说明**这种形态有跨派别的有效性** |

## 案例 4：美博城源（打首板吃面 - 渣男量教训）

| 项 | 内容 |
|---|---|
| 标的 | 美博城源 300927 |
| 日期 | 2019-09-11 |
| 操作 | **一个打首板吃面** |
| 采莲路评价 | "**只看图形感觉很好漂亮，但是这个图形为何次日容易吃面了。有兴趣的老师可以自己去复盘看看是不是渣男量**" |
| **学习要点** | **"渣男量"是采莲路的术语**——指看起来漂亮但量能有问题（如开盘瞬间放天量然后无量上板）的票。**纯看形态不够**——要看量能细节。**采莲路这种顶级游资也会吃面**——4 个点回撤的底线来自每笔小亏后立即止损 |

## 案例 5：图解 1 概览（10 案例）

| 项 | 内容 |
|---|---|
| 时段 | 2019-09 上半月 |
| 操作类型分布 | 7 次打首板 + 1 次二板 + 1 次低吸 + 1 次二次启动 |
| 胜率 | 7/10 ≈ 70% |
| **学习要点** | **采莲路高频做首板**——10 次操作 7 次是低位首板。**这是控回撤的核心**：低位首板即使失败损失也可控（不像高位接力可能直接 10% 跌） |

## 案例 6：图解 2 总结（10 案例统计）

| 项 | 内容 |
|---|---|
| 时段 | 2019-09 下半月 |
| 7 次低位首板 | **5 次吃肉，2 次吃面**（溢价都不错） |
| 1 次人气高标整理后再启动首板 | **大肉** |
| 1 次低吸 | 吃肉 |
| 1 次打二板 | 吃肉 |
| **整体胜率** | **8/10 = 80%** |
| 作者评价 | "**佩服，成功率 80% 真是比肩退神。大部分都是强势形态的打板。这才是超短的风格和爆发魅力**" |
| **学习要点** | **80% 胜率 + 强势形态打板 = 控回撤的根本**。注意：①80% 不是 100%——失败时止损要快 ②强势形态识别能力是核心壁垒 ③爆发魅力来自高胜率的复利，不是单笔暴利 |

## 怎样使用本案例库（给 Claude 自己）

B 类复盘任务流程：
1. **提取用户操作的核心要素**：是不是低位？是不是首板？是哪种形态？情绪强度？
2. **匹配 1-2 个最接近案例**
3. **判断是否符合采莲路的核心原则**：
   - ✅ 低位（30 日涨幅 < 50%）
   - ✅ 强势形态（7 种之一）
   - ✅ 形态 + 情绪双因素都满足
   - ✅ 严格控回撤
4. **回 `playbook.md` 找采莲路会怎么处理**
5. **特别提醒**：
   - 如果用户做的是接力 → 建议用一瞬流光视角
   - 如果用户追求暴利 → 提醒采莲路反对，3 年 500 倍是复利
   - 如果用户吃面后还想加仓 → 提醒采莲路严格止损，不死扛

**重要原则**：采莲路是**控回撤的极致代表**——复盘时要看用户**是不是在守住"单笔 < 2% 总资金"的回撤底线**。如果不是，即使这笔操作赚了钱，也要指出**这种操作长期会爆**。
