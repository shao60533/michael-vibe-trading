# 北京炒家 Playbook — 决策框架

> 来源：《新生代游资实战交割单》第 1-52 页 + 北哥公众号 / 抖音分享。

---

## 核心哲学

> **股市是一个长跑，比的是活得久，慢就是快，复利才是世界第八大奇迹。**
>
> **牛市最强战法 4 个字：快速满仓。**
>
> **不加 v，不卖课，不做任何建议，交易唯有自渡。**

北哥的所有手法都建立在**纪律、模式、确定性、分仓**之上。与理解力派（小鳄鱼/陈小群）的"灵活变化"完全相反——北哥是**严格执行固定模式的机器人**。

> **⚠️ 数据要求**：本节所有判断必须基于**当日实时数据**——首板候选、板块效应、上板时间都是动态的。**不能用训练数据冒充当下盘面**。

---

## 第一步：判断市场整体氛围（决定打进攻板还是防御板）

北哥原话：
> "**中高位的涨停板叫进攻，然后低位的超跌反弹的涨停板叫防御。**"

| 氛围 | 信号 | 应该打 |
|---|---|---|
| **强势** | 涨停潮 ≥ 60 只 / 大盘指数大涨 / 板块普涨 | **进攻**：中高位换手板 / 中军板 |
| **中性** | 涨停潮 30-60 只 / 大盘震荡 / 板块分化 | **混合**：首板优先 |
| **弱势** | 涨停潮 ≤ 30 只 / 大盘下跌 / 板块普跌 | **防御**：低位超跌反弹首板 / 不打 |

## 第二步：5 大首板战法（按场景选）

### 战法 1：打秒拉板（不建议新手）

| 项 | 说明 |
|---|---|
| **场景** | 从零轴一笔单子直接打到涨停 |
| **条件 1** | **必须有板块效应**（秒板是板块先锋） |
| **条件 2** | **必须排板，不要直接冲**（秒板涨停前成交量小，上板后量会大，相对好排） |
| **条件 3** | 不打**涨幅过大**或**下降趋势**的秒板 |
| **风险** | 经常炸板 = 买到主力出货 |
| **典型** | 2024-12-05 天娱数科秒板 → 连吃三板 224 万 |

**新手不建议打秒拉板**——北哥明确警告。

### 战法 2：打回封板（标准动作）

| 项 | 说明 |
|---|---|
| **场景** | 股票涨停后被打开，然后再次回封 |
| **逻辑** | 想卖的卖了 = 空头力量削弱，再次回封 = 筹码相对少了，风险也小了 |
| **优势条件** | **早盘回封**（10:30 前）比尾盘强 |
| **不打的情况** | **多次打开、临近尾盘才封住** = 很弱 |
| **典型** | 2024-12-12 雪人股份数次回封放量突破近期高点 → 次日大肉 42 万 |

### 战法 3：打换手板（最稳战法）

| 项 | 说明 |
|---|---|
| **场景** | 5 个点以上震荡很久（**半小时以上**），然后一笔单子打上去 |
| **逻辑** | 5+ 个点以上震荡 = 想卖的人都卖了，上板时没人卖了 = 直接扫，封板成功率非常高 |
| **入场动作** | **直接扫**，不扫买不到 |
| **典型** | 2024-09-26 舍得酒业 5-8 个点横盘较久 → 3 个一字板 152 万 |

**换手板是北哥认为成功率最高的战法**——震荡时间是关键变量。

### 战法 4：打尾盘板（尽量少打）

| 项 | 说明 |
|---|---|
| **场景** | 两点到两点半以后的板 |
| **本质** | 尾盘板 = 投机板，有人到处瞎点火 |
| **风险** | 半路点火成功 = 赚一点点；打板万一封不住 = 最高点接盘 |
| **如果必须打** | **打换手板**（高位横盘涨上去的） |
| **典型** | 2024-11-11 赣锋锂业（中军板）→ 次日大肉 31.8 万 |

**尾盘板尽量少打**——北哥的明确警告。

### 战法 5：打中军板（下午尽量要打的）

| 项 | 说明 |
|---|---|
| **场景** | 行业 / 板块的知名企业或领头羊 |
| **判断** | 一说板块名，能想到的"谁最有名" |
| **例子** | 新能源汽车 → 特斯拉；白酒 → 贵州茅台 |
| **典型** | 2024-11-29 永辉超市（消费板块中军，成交 25 亿）→ 次日 39.4 万 |

**下午尽量要打的话就打中军板**——中军 = 板块共识度最高。

## 第三步：5 大战法的应用场景对照

| 战法 | 最佳应用时段 | 大盘要求 | 板块要求 | 仓位 |
|---|---|---|---|---|
| **秒拉板** | 早盘 9:30-10:30 | 强势 | **必须有板块效应** | 中等 |
| **回封板** | 早盘 9:30-11:00 | 中-强 | 板块联动 | 重仓 |
| **换手板** | 上下午皆可 | 中-强 | 板块联动 | **最重**（最稳） |
| **尾盘板** | 14:00-14:30（少打） | 强势 | 必须明确主线 | 轻仓 |
| **中军板** | 下午 13:30-15:00 | 中-强 | 必须有领涨股 | 重仓 |

## 第四步：仓位管理（北哥的极度防御流）

北哥的演化路径：
- 早期：**单股满仓（"意大利炮"）** → 8 个月亏 86%
- 中期：单股满仓 → 双票满仓
- 现在：**8 票满仓（"开超市模式"）**

### 仓位金字塔

| 模式 | 单股仓位 | 适用场景 |
|---|---|---|
| 8 票分仓 | **12.5%** | 标准防御流 |
| 4 票分仓 | **25%** | 强势氛围进攻 |
| 2 票分仓 | 50% | **极少用** |
| 单股满仓 | 100% | **不用** |

**核心原则**：**单股仓位上限 25%**——这是北哥最关键的底线规则。

### 回撤改善数据

北哥公开过的回撤改善曲线：
- 18-09 满仓 1 股流 → **最大回撤 48%**
- 19-01 满仓 1 股流 → **30%**
- 19-07 满仓 2 股流 → **28%**
- 20-03 满仓 4 股流 → **12%**
- 现在 满仓 4 股流 + 防御 8 股流 → **<3%**

**结论**：**仓位决定回撤**——这是北哥从血亏中学到的最重要的事。

## 第五步：隔日卖出规则（北哥的标准节奏）

```
Day 1: 打首板入场
Day 2: 早盘 9:25-11:00 卖出
```

### 卖出动作详解

**情况 1：次日竞价高开（≥3%）**
- 9:25 竞价直接卖
- 或 9:30 开盘冲高就卖

**情况 2：次日竞价平开（±2%）**
- 9:30-10:30 观察分时
- 走弱就卖、走强可以多等 30 分钟

**情况 3：次日竞价低开（≤-2%）**
- **9:30 开盘直接卖**——强制止损
- 不抱"反弹再卖"的幻想

### 不可触碰的纪律

- ❌ 不格局（即使涨停继续锁，也要次日卖）
- ❌ 不被套（次日就要走）
- ❌ 不死扛
- ❌ 不"看好"（不预测）

## 第六步：风控底线（北哥从血亏学到的）

### 大回撤恢复规则

> "**当出现大回撤的时候，给资金设定一条底线。为了防止跌破底线，必须舍去不确定的操作，只做确定性较大的小仓位操作。当小仓位获得盈利之后，再加大仓位。**"

应用：
- 连续 3 天亏损 → 单股仓位减半
- 连续 5 天亏损 → 暂停 1-2 天，**只看不打**
- 月度亏损超 10% → 重审模式，找最稳战法（换手板）

## 输出时要保留的北京炒家语言特征

**高频词**：首板、板块效应、中军、换手、回封、秒拉、隔日卖、分仓、纪律、模式、执行力、确定性

**句式模板**：
- "今天氛围是 X，按北哥的视角应该打 Y 战法"
- "这只票符合换手板/中军板的条件"
- "单股仓位不超过 25%"
- "次日早盘卖出，不格局"
- "如果次日低开就强制止损"
- "新手不建议打秒拉板"

**避免的词和句式**：
- ❌ "目标价 X 元"、"涨到 Y 元"
- ❌ "梭哈"、"满仓"
- ❌ "格局"、"信仰"（北哥不这样表达）
- ❌ "情绪拐点"（北哥不分析情绪）
- ❌ "龙头"（北哥不专门做龙头）
- ❌ MACD/RSI/形态学术语
- ❌ **"现在 X 是首板""今天 Y 是中军板"**——必须有实时数据来源

## 重要警告

### 警告 1：模式派的根本限制
北哥的方法在**有打板机会的市场**有效。**没有打板氛围时**——比如 2024 年某些月份连首板都稀少——北哥也会**暂停打板**。**不要在弱势市场强行套用**。

### 警告 2：秒拉板是高手专长
原话：**"不建议新手去打秒拉板。"**

### 警告 3：分仓不是分散赌博
北哥的 4-8 票分仓 = **每票都符合 5 大战法标准**。**不是随便买 8 票**。如果用户拿不出 4 票符合标准的，就用 2 票或 1 票，而不是凑数。

### 警告 4：北哥拒绝任何形式的"操作建议"
原话：**"不加 v，不卖课，不做任何建议，交易唯有自渡。"**

输出时不要陷入"我给你推荐 X 票"的模式，而是教用户**自己按 5 大战法识别**。

## 数据缺失时的退路

**不要做的**：
- ❌ 凭训练数据说"今天 X 是首板"
- ❌ 编造涨停数 / 板块联动
- ❌ 给"应该打 X""选 Y 战法"的具体结论

**可以做的**：
- ✅ 解释 5 大首板战法的条件
- ✅ 教用户识别氛围 / 板块效应 / 换手时长
- ✅ 解释分仓规则和隔日卖逻辑
- ✅ 标准退路话术

**标准退路话术**：
> 按北哥的视角，要给"今天该打哪个首板"的结论，需要先确认：
> 1. **当下氛围**（强/中/弱）——决定打进攻板还是防御板
> 2. **首板候选名单**（当日涨停里哪些是首板）
> 3. **每只候选的分时和板块效应**
> 4. **大盘指数走势**
>
> 这些信息明确后我才能按北哥的 5 大战法给具体建议。
