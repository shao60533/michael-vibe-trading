## 案例 1：天娱数科（战法 1 - 秒拉板教科书）

| 项 | 内容 |
|---|---|
| 标的 | 天娱数科 002354 |
| 日期 | 2024-12-05 |
| 主线 | 人工智能（OpenAI 未来 12 天密集展示新品；海外 AI 应用连续大涨） |
| 战法 | **秒拉板** |
| 板块效应 | 当日人工智能板块 17 只 252 亿成交 |
| 入场判断 | 当日华扬联众快速秒板大单封住；龙头利欧股份在昨天爆量 137 亿后开盘也是迅速从平盘拉起冲击七板；在确定板块强度后，北哥才打了快速上板的天娱数科 |
| 入场动作 | **打秒拉板**（严重缩量的票 = 相对好排） |
| 持仓 | **连吃三板** |
| 结果 | **斩获 224 万** |
| 北哥评价 | "**这种票非常考验看盘能力，要结合当日情绪，板块强度，还要盯到、买到，基本都是一瞬间决定**" |
| **学习要点** | **秒拉板的 3 个前提条件全部具备**：①板块龙头确立（利欧七板）②板块联动充分（17 只）③目标票严重缩量（好排板）。**新手不要学**——这要求看盘能力极强 |

## 案例 2：雪人股份（战法 2 - 回封板教科书）

| 项 | 内容 |
|---|---|
| 标的 | 雪人股份 002639 |
| 日期 | 2024-12-12 |
| 主线 | 冰雪产业（前晚出利好） |
| 战法 | **回封板** |
| 板块联动 | 冰山冷热秒拉板 → 莱茵体育一进二封板 → 雪人股份数次回封 |
| 入场判断 | 开盘并没有资金冲进来；10:33 分发动；在确定板块效应后，才打了**数次回封放量突破近期高点的雪人股份** |
| 入场动作 | 打回封板（突破近期新高 + 量能超过前期高点换手充分） |
| 持仓 | 1 天 |
| 结果 | **次日大肉 42 万多走人** |
| **学习要点** | **回封板的入场顺序**：①等板块龙头确立（不要先打）②看到板块联动后再打 ③选有"突破近期高点 + 量能放大"特征的回封票。**北哥不打"开盘就冲进来"的票**——他要先看板块发动情况 |

## 案例 3：舍得酒业（战法 3 - 换手板教科书）

| 项 | 内容 |
|---|---|
| 标的 | 舍得酒业 600702 |
| 日期 | 2024-09-26 |
| 主线 | 消费板块（白酒）— 历年主攻点之一 |
| 战法 | **换手板** |
| 板块联动 | 当日白酒 16 只 224 亿成交（贵州茅台 3 天涨超 20%；行业国庆旺季将至） |
| 入场判断 | 北京炒家打了 **5 到 8 个点横盘较久的舍得酒业** |
| 入场动作 | 直接扫板（横盘充分换手后上板） |
| 持仓 | **三个一字板** |
| 结果 | **到手 152 万** |
| **学习要点** | **换手板是北哥认为成功率最高的战法**——核心变量是**横盘时长**。5-8 个点震荡半小时以上 = 想卖的都卖了 = 上板时直接扫，封板成功率非常高。**消费板块是历年主攻点**——这是北哥对历史规律的归纳 |

## 案例 4：赣锋锂业（战法 4/5 - 尾盘中军板）

| 项 | 内容 |
|---|---|
| 标的 | 赣锋锂业 002460 |
| 日期 | 2024-11-11 |
| 主线 | 锂电（赣锋是绝对的行业中军） |
| 战法 | **尾盘板 + 中军板**（重叠） |
| 入场判断 | 当天大盘一路放量上攻；尾盘北京炒家打了**横盘已久的赣锋锂业**；**也可以算中军板** |
| 入场动作 | 尾盘打中军板（虽然北哥说"尾盘板尽量少打"，但中军板是例外） |
| 持仓 | 1 天 |
| 结果 | **次日大肉 31.8 万** |
| **学习要点** | **尾盘板的唯一安全选择 = 中军板**。北哥的逻辑：**"下午尽量要打的话就打这些行业的中军"**——中军板的封板概率远高于一般尾盘板，因为中军是板块共识度最高的票 |

## 案例 5：永辉超市（战法 5 - 中军板教科书）

| 项 | 内容 |
|---|---|
| 标的 | 永辉超市 601933 |
| 日期 | 2024-11-29 |
| 主线 | 消费板块大爆发 |
| 战法 | **中军板** |
| 板块联动 | 当日消费 26 只 132 亿成交 |
| 入场判断 | 北京炒家打了**板块中军**，**成交 25 亿的容量票** |
| 入场动作 | 打中军板（容量大 = 资金可承载） |
| 持仓 | 1 天 |
| 结果 | **次日收获 39.4 万走人** |
| **学习要点** | **大资金做打板必选大容量中军板**——北哥的资金量级决定了不能打小盘票。**中军板 = 板块共识 + 资金容量** 双重保险 |

## 案例 6：早期完整成交流水（2019-02 至 2019-05）

| 项 | 内容 |

| 操作模式 | 标准首板节奏（单股仓位 50-150 万） |

## 怎样使用本案例库（给 Claude 自己）

B 类复盘任务流程：
1. **提取用户操作的核心要素**：是不是首板？哪种战法（秒拉/回封/换手/尾盘/中军）？仓位多少？持仓多少天？
2. **匹配 1-2 个最接近案例**
3. **判断是否符合该战法的入场条件**（5 大战法每个都有明确条件）
4. **特别关注违反北哥纪律的操作**：
   - 高位板 / 二板接力 → 北哥不做
   - 单股满仓 → 北哥的早期教训
   - 格局持仓 → 北哥的标准节奏是隔日卖
5. **回 `playbook.md` 找北哥会怎么处理**

**重要原则**：北哥的纪律是**铁律**——分仓 + 隔日卖 + 战法识别。复盘时如果用户的操作违反任何一条，要直接指出（即使这笔操作赚了钱）。**模式派的核心是纪律，纪律破一次就是 8 个月血亏的开始**。
