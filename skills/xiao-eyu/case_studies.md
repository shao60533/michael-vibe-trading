## 第一组：教科书操作期（2014 年 12 月，券商主线）

> **背景**：2014 年底牛市启动，券商是最强主线。小鳄鱼围绕中信证券（板块中军龙头）做了一系列经典操作。

### 案例 1：中信证券接力（核心权重 / 主线启动期）

| 项 | 内容 |
|---|---|
| 标的 | 中信证券 600030 |
| 日期 | 2014-12-05 |
| 主线 | 券商（当时最强主线） |
| 位置 | **核心权重 / 中军龙头** |
| 节奏 | 启动期 |
| 操作 | 接力买入 22.47 |
| 结果 | 12-08 卖出 25.14（次日大肉） |
| 小鳄鱼自评 | "做的是核心权重中军的接力，大肉" |
| **学习要点** | **指数、热点主流、节奏**——三件套对齐时，核心权重接力是最高确定性玩法 |

### 案例 2：中信证券卖飞追回（卖飞处理）

| 项 | 内容 |
|---|---|
| 标的 | 中信证券 600030 |
| 日期 | 2014-12-08 当天再买回 25.49 |
| 节奏 | 主升期 |
| 操作 | 当天卖飞后追回 |
| 结果 | 12-09 卖出 27.65（继续大肉） |
| 小鳄鱼自评 | "强势主线确实容易卖飞，**一旦卖飞丧失先手，要追回来还是比较被动**，除非当天这个方向情绪依旧非常强" |
| **学习要点** | 卖飞后追回的**前提条件**是"当天该方向情绪依旧非常强"。如果情绪不强，**不追**（这是大多数情况） |

### 案例 3：营口港打板吃面（主线外杂毛 / 节奏判断错）

| 项 | 内容 |
|---|---|
| 标的 | 营口港 600317（已退市） |
| 日期 | 2014-12-09 |
| 主线 | **不在券商主线里** |
| 位置 | 人气老牛股横盘再冲板 |
| 操作 | 扫板（或炸板低吸）买入 5.17 |
| 结果 | 12-10 卖出 5.21（小幅卖飞 / 失败） |
| 小鳄鱼自评 | "次日开盘分时不够强，零轴附近离场，当日小卖飞。**如果入场逻辑是打板，这种非当下主线，次日分时不强确实不敢拿**" |
| **学习要点** | **不在当下主线里的票**，即使有人气历史，次日分时不强就要果断离场——不要因为"已经涨了七八倍"就期待还有空间 |

### 案例 4：长江证券首阴低吸（最完美低吸点）

| 项 | 内容 |
|---|---|
| 标的 | 长江证券 000783 |
| 日期 | 2014-12-10 |
| 主线 | 券商（仍是最强主线） |
| 位置 | 主线龙头首阴后的低吸 |
| 节奏 | **首阴期** |
| 操作 | 低吸买入 13.24（几乎是当日最低点） |
| 结果 | 12-12 卖出 15.19（**两天十几个点的大肉**） |
| 小鳄鱼自评 | "**这应该是主线最好的低吸点**，激进一点的可能前一天首阴就进去了，但是这么一个大板块，又是大票，高位大阴没必要这么心急" |
| **学习要点** | **首阴次日低吸是主线最完美的入场点**——主线没死，只是龙头需要换手，胜率远高于追涨。注意"次日"而不是"首阴当天"——主线越大、票越大，越不急 |

### 案例 5：国海证券后排低吸（防守仓位）

| 项 | 内容 |
|---|---|
| 标的 | 国海证券 000750 |
| 日期 | 2014-12-15 |
| 主线 | 券商 |
| 位置 | **主线后排**（涨幅落后、先于调整） |
| 操作 | 低吸买入 16.86 |
| 结果 | 12-16 卖出 29.65（次日大涨但小鳄鱼卖飞了大头） |
| 小鳄鱼自评 | "**后排的低吸，次日卖飞**……这个时候去做偏防守的低吸也是不错，如果继续强这种后排可能强势补涨" |
| **学习要点** | **后排低吸是防守仓位**——如果主线继续强，后排补涨；如果主线转弱，先调整过的后排反而抗跌。但要接受"卖飞大头"的常态——后排不会有龙头那么强的节奏感 |

### 案例 6：再次接力中信证券中军（高低切换）

| 项 | 内容 |
|---|---|
| 标的 | 中信证券 600030 |
| 日期 | 2014-12-16（卖出 12-17 价格 31.92） |
| 主线 | 券商 |
| 位置 | 核心权重 / 中军 |
| 节奏 | **转强期** |
| 操作 | 卖国海换中信（高低切到龙头） |
| 结果 | 大肉（12-17 卖 31.92） |
| 小鳄鱼自评 | "当天中军带领券商继续走强，小鳄鱼换到最强中军上……**龙头的溢价更高**。就好比最近陈小群看到剑桥带领光模块拉伸，马上减仓新易切换部分到龙头剑桥上，大赚 20 多个点" |
| **学习要点** | **主线转强信号出现时，要把仓位往最强龙头集中**——后排可以放手，龙头的溢价永远更高。这是理解力派的"高低切"动作 |

### 案例 7：民生银行衍生分支接力（吃面）

| 项 | 内容 |
|---|---|
| 标的 | 民生银行 600016 |
| 日期 | 2014-12-18 |
| 主线 | 券商（开始走弱） |
| 位置 | **主线衍生分支**（券商调整，资金试图切到银行） |
| 操作 | 接力买入 9.89 |
| 结果 | 当日卖出 9.40（**当日吃面**，分支没打出高度） |
| 小鳄鱼自评 | "可能当时市场有资金往涨幅更小的大金融分支银行上切，但是没打出高度，小鳄鱼接力吃面。**逻辑也没大问题**，从走势看后面几天民生还在慢涨" |
| **学习要点** | **衍生分支的接力，逻辑对不代表能赚到钱**——分支必须当天打出高度，否则就是吃面。这是理解力派的常见失误，**事后看逻辑没错，但短线交易必须看节奏** |

### 案例 8：中铁工业主线外人气股打板（追 6 板的大肉）

| 项 | 内容 |
|---|---|
| 标的 | 中铁二局/中铁工业 600528 |
| 日期 | 2014-12-19 |
| 主线 | **不在券商主线里**，是基建/铁路独立人气 |
| 位置 | **主线外的人气强势股** |
| 节奏 | 主线（券商）走弱时的资金转移 |
| 操作 | 打板买入 14.49（追了 6 板） |
| 结果 | 12-22 卖出 15.94（炸板大肉离场） |
| 小鳄鱼自评 | "当一波大主线高位走弱，此时低位人气股（刚走第一波人气拉伸）容易得到资金拥趸。**小鳄鱼虽然追了 6 板，但依旧大肉**" |
| **学习要点** | **当主流主线走弱时，要找"低位刚启动的新人气股"**——不要死守已经走完的主线。这种切换是理解力派最考验"快速感知"的地方 |

### 第一组小结

> 本期 7 次大肉，1 次吃面（民生银行衍生分支）。
> 小鳄鱼原话："本期内容学会且会应用，足够你成为顶级游资，**因为这就是当下最强手法核心**。"
> **核心模式**：围绕主线，做核心权重接力 → 龙头首阴低吸 → 转强换龙头 → 主线走弱时切人气股

## 第二组：莱茵生物 7 天龙头持仓复盘（2014 年 8 月）

> **背景**：演示了"龙头持仓 + 做 T"的细节，重点是"卖飞"的反思

### 案例 9-14（前 6 天，按日操作）

| 天 | 操作 | 结果 |
|---|---|---|
| 第 1 天 | 8-05 10:42 买入 18.68 | 入场 |
| 第 2 天 | 8-06 卖出 19.75 / 再买入 19.77 / 又卖出 19.66 | 频繁做 T，吃肉但折腾 |
| 第 3-6 天 | 持仓做 T | 多次卖飞、追回 |

（具体每日明细在 markdown 笔记 P.219-222）

### 案例 15：第七天的总结（持仓心态反思）

| 项 | 内容 |
|---|---|
| 标的 | 莱茵生物 |
| 节奏 | 龙头连续转强 |
| 操作问题 | "在两个转强点卖飞多" |
| 小鳄鱼自评（原话） | "这种没有绝对错，如果你想把握每个分时波动的话，但纵观整体**还不如拿着不动**，越往上 T 你越不敢上仓位，至少我是这样，小鳄鱼也是这样。**所以拿住龙头真的很考验心态**" |
| **学习要点** | **龙头持仓时少做 T**——每次做 T 都会降低你的仓位（"越往上 T 越不敢上仓位"），最终拿不住大肉。**真正的纪律不是技术，是心态**：相信主线、相信龙头，让仓位跑完节奏 |

### 第二组总结

> **围绕龙头做总是吃肉多，卖飞多；杂毛则吃面多，被套多。**
>
> 这是理解力派的核心权衡——选择"卖飞的痛苦"而不是"吃面的痛苦"

## 第三组：职业生涯最差期（2014 年 7-8 月）

> **背景**：小鳄鱼自己公开发帖说这是**"职业生涯最差的交易"**。市场大盘低迷后开始拉伸，但**没有清晰主线**，资金全都走趋势。小鳄鱼分仓打板模式失效，"打啥炸啥"。

### 案例 16-27（共 12 个图，全部炸板/吃面）

| # | 标的 | 操作 | 结果 |
|---|---|---|---|
| 16 | 002091 | 强势图形竞价开盘扫板 | 离场未知 |
| 17 | 600028 | 三板打入 | 主线不明 / 吃面风险高 |
| 18 | 三木集团 000061 | 同类型 | 题材判断难 |
| 19 | 福建高速 600033 | **扫首板，炸的稀碎** | 吃面 |
| 20 | 平潭发展 000182 | 二板炸进去 | 离场未知 |
| 21 | 新疆能源 000969 | 排板炸进入首板 | 吃面 |
| 22 | 华纺股份 000509 | 首板扫，炸的稀碎 | 吃面 |
| 23 | 啤酒股份 002550 | 竞价顶一字，炸的稀碎 | 吃面 |
| 24 | 龙江交通 601188 | 扫首板 | 离场未知 |
| 25 | 中国铝业 601600 | 打首板 | 离场未知 |
| 26 | 陕西煤业 601225 | 排到首板，炸的稀碎 | 吃面 |
| 27 | 大有能源 600403 | 强势形态再启动冲板扫，炸的稀碎 | 吃面 |

### 关键反思（书内 P.229 原话）

> "**小鳄鱼当时真是打啥炸啥**，正如他贴子所说，职业生涯最差。从当时大盘走势来看还挺猛。我看他打的这些也是和指数同步拉伸，**是不是有一条更强的主线抽血，还是没主线全都走趋势，反而没资金去接力**。"
>
> "**市场主流资金都在围绕主线走趋势为主，所以能否快速应对市场的变化才是最重要的**。"

### 关键反思（章末原话）

> "市场是变化的，核心在于你有没有快速感知到市场的这种变化，并作出应对。"

### 第三组学习要点（极其重要）

1. **没有清晰主线时不要分仓打板**——这是顶级游资也会犯的错
2. **指数涨不代表你能赚到钱**——如果资金都在走趋势而不接力，打板模式会失效
3. **要主动识别"市场风格切换"**——从连板模式切到趋势模式时，所有打板手法都要重审
4. **小鳄鱼自己改正后的纪律**：
   - **几乎不排板**——"以前排是自己没时间看盘，又怕错过，干脆就排着，结果走扎进去的都是吃面多"
   - **几乎不扫板**——宁愿打回封
   - **炸了先观望**——"看承接，看量，看炸了后排的表现是回落还是依旧走强，再决定是否打入"

## 案例匹配速查表（B 类复盘任务用）

## 怎样使用本案例库（给 Claude 自己）

收到 B 类复盘任务时：

1. **提取用户操作的核心要素**：标的、入场逻辑、节奏、是否在主线里
2. **按速查表匹配最接近的 1-2 个历史案例**
3. **读对应案例的"小鳄鱼自评"和"学习要点"**
4. **回到 `playbook.md` 找小鳄鱼会怎么做这笔交易**
5. **输出时引用案例号**（如"这跟案例 4 长江证券的首阴低吸模式很像"），让用户能追溯

记住：**理解力派不评判用户的具体盈亏，只评判判断框架是否对**。即使用户这笔交易赚了大钱，如果框架是"运气好"，也要指出来；即使用户亏了，如果框架对、只是节奏判断错，要肯定框架。
