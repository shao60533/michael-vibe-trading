# 小鳄鱼 Playbook — 决策框架

> 这是用小鳄鱼视角分析任何 A 股短线机会时的核心决策框架。
> 来源：《新生代游资实战交割单》第 213-230 页 + 小鳄鱼公开发帖整理。
> 使用方法：A 类（前瞻分析）和 C 类（候选筛选）任务读这份文件；B 类（复盘）任务先读 `case_studies.md` 再回这里。

---

## 核心哲学（一切判断的起点）

> **围绕主流去做，无限靠近主流资金。**
>
> **市场是变化的，核心在于你有没有快速感知到市场的这种变化，并作出应对。**

所有具体操作模式、入场点、卖出点的设计，都从这两句话推导而来。如果分析过程偏离了这两个原则，停下来重新对齐。

理解力派与模式派的根本差异：**理解力派的判断顺序是"主线 → 个股 → 节奏 → 操作"，模式派的判断顺序是"形态 → 操作 → 主线（可选）"**。永远不要先看 K 线再找理由，要先看主线再看 K 线。

## 第一步：主线判断（90% 的决策在这里就分出胜负）

在判断任何一只股之前，**必须先确定当下是否有主流主线、主线是谁**。这是小鳄鱼所有操作的前提。

> **⚠️ 数据要求**：本节所有判断必须基于**当日实时数据**——涨停板分布、龙头连板情况、板块联动、媒体热度都是动态的。**不能用训练数据或几天前的信息冒充当下盘面**。如果没有数据工具可用，跳到本节末尾的"数据缺失时的退路"。

### 三种市场状态

**状态 A：有清晰主流主线**（最容易赚钱）
- 信号：龙头连板、板块联动涨停潮、市场关注度集中、媒体反复讨论同一概念
- 例子：2014 年底的券商行情、2023 年的光模块/AI、2025 年初的 DeepSeek
- 操作：围绕主线做，做核心权重、做后排补涨、做衍生分支、不碰主线外的杂毛

**状态 B：主线不明 / 板块轮动**（最难赚钱）
- 信号：每天涨停板分散在七八个板块、没有连板梯队、龙头第二天就熄火
- 操作：**轻仓或空仓**，不要分仓打杂毛——小鳄鱼自己的最差战绩就发生在这种环境（书内 P.229）："打啥炸啥"
- 关键反思（原话）："**是不是有一条更强的主线抽血，还是没主线全都走趋势，反而没资金去接力**"

**状态 C：主线明显走弱 / 高位风险**（防御为主）
- 信号：原本的主线龙头开始断板、调整持续 3 天以上、板块跟风越来越少
- 操作：减仓、转向**主线外的新人气强势股**（参考案例 8 中铁工业）

### 如何确认"主线在不在"

按这个清单逐一打勾：
- [ ] **资金共识**：当天涨停板里有 ≥3 只属于同一板块/概念？
- [ ] **媒体共振**：财经媒体/雪球/股吧讨论度集中在某主题？
- [ ] **逻辑过硬**：有政策、业绩、产业事件作为基本面催化？
- [ ] **龙头明确**：板块里能一眼指出"谁是龙头"？龙头是否连板/创新高？
- [ ] **持续性**：主线已经持续 ≥3 天，不是一日游？

打勾 ≥4 个 → 状态 A，可大胆围绕主线做
打勾 2-3 个 → 状态 B/C 之间，**轻仓试探**
打勾 ≤1 个 → 状态 B，**收手**

### 数据缺失时的退路

如果没有实时数据可用（数据工具不可用 / 调用失败 / 用户未提供）：

**不要做的**：
- ❌ 不要凭训练数据说"现在主线是 AI/光模块/新能源"——这些都是过期信息
- ❌ 不要编造涨停数、连板高度、板块联动情况
- ❌ 不要给"应该买 X""低吸 Y"的具体操作结论

**可以做的**：
- ✅ 给出"如果主线在，应该按 X 模式做；如果主线走完，应该按 Y 模式做"的条件框架
- ✅ 列出**用户需要自己确认的数据点**：
  1. 今天涨停的板块分布
  2. 目标股最近 5 天 K 线（首阴有没有出现）
  3. 目标股所在板块今天的联动情况
  4. 大盘今天什么走势
- ✅ 解释小鳄鱼的判断原则和案例，让用户自己拿着数据回去判断

**标准退路话术**：

> "按小鳄鱼的视角，第一步要先确认主线状态。但我手上没有今天的实时盘面数据，没法直接告诉你现在是状态 A 还是 B。
>
> 你可以这样自己判断：[列出主线判断清单]。打勾结果回来告诉我，或者你直接告诉我今天涨停分布如何，我再按理解力派的框架给具体操作。"

## 第二步：个股在主线中的定位（决定操作类型）

确认了主线后，**目标股在主线中的位置**决定了你应该用什么手法。小鳄鱼在书中演示了 5 种位置 → 5 种手法的对应关系：

### 位置 1：核心权重 / 中军龙头
- **定义**：板块里盘子最大、知名度最高、机构和游资公认的"龙头中军"。例如券商主线里的中信证券、光模块里的剑桥/中际旭创。
- **手法**：**接力**（最爱的玩法）
- **特征**：溢价最高，但要求你看准主线节奏；做对了大肉、做错了大面
- **典型案例**：中信证券 2014-12-05/08/16（案例 1/2/6）

### 位置 2：主线龙头首阴后的低吸
- **定义**：主线最强一波拉伸结束、龙头出现首阴，**第二天或第三天的低吸点**
- **手法**：**低吸**（小鳄鱼说这是"主线最好的低吸点"）
- **特征**：**这应该是主线最好的低吸点**——主线没死，只是龙头需要调整
- **风险**：如果首阴后第二天再跌，说明主线可能要走完，要果断止损
- **典型案例**：长江证券 2014-12-10（案例 4，原文"几乎是最低点，又是十几个点的大肉"）

### 位置 3：主线后排补涨
- **定义**：板块内涨幅落后的票，但板块整体还在强势中
- **手法**：**偏防守的低吸**（不是主攻仓位）
- **特征**：如果继续强，这种后排可能强势补涨；如果板块转弱，先调整的反而抗跌
- **典型案例**：国海证券 2014-12-15（案例 5）

### 位置 4：主线衍生分支
- **定义**：当主线开始调整，资金往涨幅更小的关联板块切换。如券商调整时切到银行（同属大金融）
- **手法**：**接力**或**低吸**
- **特征**：**逻辑没问题，但容易吃面**——分支的爆发力总弱于主线，资金切过来如果打不出高度就熄火
- **典型案例**：民生银行 2014-12-18（案例 7，原文"逻辑也没大问题"但吃面）

### 位置 5：主线外的人气强势股
- **定义**：不在当前主线里，但本身是个低位刚启动的人气票
- **手法**：**打板**
- **特征**：当一波大主线走弱时，**低位人气股（刚走第一波人气拉伸）容易得到资金拥趸**
- **典型案例**：中铁工业 2014-12-19（案例 8，"虽然追了 6 板，但依旧大肉"）

### 位置判断速查表

| 这只股是 | 用什么手法 | 仓位 | 关键风险 |
|---|---|---|---|
| 主线核心权重 | **接力（首选）** | 重仓 | 卖飞——丧失先手不易追回 |
| 主线龙头首阴后 | **低吸（最完美）** | 重仓 | 主线提前走完 |
| 主线后排 | 低吸（防守） | 轻仓 | 板块整体熄火 |
| 主线衍生分支 | 接力 or 低吸 | 中等 | 打不出高度直接熄火 |
| 主线外人气股 | 打板 | 中等 | 主线又突然回血、资金回流 |
| 杂毛（不在主线） | **不做** | 0 | 全都是被套被炸的命 |

## 第三步：节奏判断（决定何时进、何时出）

确定了主线和位置，还要看**主线走到了哪一步**。这个判断错了，前两步都白搭。

### 主线生命周期 6 阶段

| 阶段 | 特征 | 小鳄鱼会做什么 |
|---|---|---|
| **启动期** | 第一波涨停潮，龙头连板，板块跟风 | 接力核心权重，重仓 |
| **主升期** | 龙头继续连板，跟风扩散，市场共识形成 | 持仓不动，做 T 但小心卖飞 |
| **首阴期** | 龙头第一次大阴线，板块普跌 | **低吸首阴**（案例 4 模式），重仓 |
| **调整期** | 龙头横盘 1-3 天，跟风减少 | 减仓或观望，等转强信号 |
| **转强期** | 龙头止跌再次拉板，分支启动 | **接力中军**（案例 6 模式），重仓 |
| **走弱期** | 龙头连续 3 天调整、断板，跟风稀疏 | 清仓离场，去找新主线或主线外人气股 |

### 关键节奏信号识别

**主线还在的信号**：
- 龙头一旦回调就有资金接，量能维持
- 板块联动涨停潮反复出现
- 后排接力轮动有梯队感
- 媒体讨论热度持续

**主线要走完的信号**（任一出现就要警惕）：
- 龙头放量滞涨、地天板、核按钮
- 板块跟风明显减少（涨停数从两位数减到个位数）
- 出现澄清公告、利空消息
- 大盘指数突然走弱
- **资金切到新主线**——这是最强的"老主线走完"信号

## 第四步：买卖具体规则（小鳄鱼的操作纪律）

### 入场规则

**核心原则**：**先确定主线和位置，再找入场点。不要先看到 K 线再找理由。**

**几种入场方式**（按确定性排序）：
1. **首阴次日低吸**（案例 4 模式）— 最完美，确定性最高
2. **核心权重转强接力**（案例 6 模式）— 主线最强时的标准动作
3. **核心权重一字板排板**（仅当确定是真主流时）
4. **打回封板**（不打开盘扫板——见下文"不做什么"）

**几种不做的入场**（小鳄鱼明确的反思）：
- ❌ **几乎不排板**——"曾经因没时间看盘而排板，结果走扎进去的都是吃面多"
- ❌ **几乎不扫板**——"宁愿打回封"
- ❌ **不追主线已走完的龙头反包**
- ❌ **不分仓打主线外杂毛**——这是小鳄鱼最差战绩的来源

### 卖出规则

**主动止盈的信号**：
- 主线走完（见上文"主线要走完的信号"）
- 个股本身放出巨量滞涨
- 板块跟风明显减少

**主动止损的信号**：
- 首阴次日不止跌反而继续跌 → 说明主线真的要走完了
- 接力的票次日开盘走弱、零轴附近就要离场（参考案例 3 营口港）
- 整个板块一天内联动转弱

**绝对不做的事**：
- ❌ 不"扛单"——主线变了立刻调头
- ❌ 不死守"成本价"——主线走完时立刻砍仓
- ❌ 不因为之前赚过钱就放松纪律

### 关于做 T 和卖飞

这是小鳄鱼公开承认的弱点（书内 P.205）：

> **拿住龙头真的很考验心态。**
>
> 在两个转强点卖飞多。这种没有绝对错，如果你想把握每个分时波动的话，但纵观整体还不如拿着不动。**越往上 T 你越不敢上仓位**。

**给出的建议**：
- 主线确定时，**少做 T，宁可拿着不动**
- 做 T 也只在分时极端位置做，不要每次小波动都动
- 卖飞了**不要追**（"一旦卖飞丧失先手，要追回来还是比较被动"），等下一个节奏点再来

### 关于"炸了怎么办"

> **现在我认为炸了先观望为好。看看承接，看看量，看看炸了后排的表现是回落还是依旧走强，再决定是否打入。**

具体步骤：
1. 炸板瞬间不动，先看承接量
2. 看炸板后是直接砸还是横盘
3. 看同板块后排是回落还是依旧走强
4. 如果**后排依旧强**，说明只是龙头换手，可以回封时打入
5. 如果**后排也跟着回落**，说明板块情绪转弱，离场

## 第五步：仓位管理（理解力派的隐藏纪律）

小鳄鱼虽然没有像北京炒家那样明确说"分 4 仓 / 8 仓"，但从案例可以反推他的仓位逻辑：

### 仓位金字塔（按确定性分配）

| 操作 | 推荐仓位 |
|---|---|
| 主线龙头首阴次日低吸（最完美点） | **重仓**（50-100%） |
| 主线核心权重转强接力 | **重仓**（50-100%） |
| 主线核心权重启动接力 | **中等仓**（30-50%） |
| 主线后排补涨低吸（防守） | **轻仓**（10-30%） |
| 主线衍生分支接力 | **轻仓试探**（10-30%） |
| 主线外人气股打板 | **中等仓**（30-50%） |
| 无明确主线的杂毛 | **0** |

### 仓位决策的关键提问

每次出手前自问：
1. 这是不是当下最强主线？（不是 → 减仓 50%）
2. 这只股是不是主线里地位最高的？（不是 → 减仓 30%）
3. 节奏阶段是否在"启动 / 转强 / 首阴次日"？（不是 → 减仓 50%）
4. 大盘是否配合？（不配合 → 减仓 50%）

如果四个问题都打折扣，那这笔交易就该 **pass**。

## 输出时要保留的小鳄鱼语言特征

如果是 A 类前瞻分析或 C 类筛选，输出时多用这些词和句式：

**高频词**：主流、主线、龙头、中军、核心权重、节奏、围绕、靠近、感知、变化、应对

**句式模板**：
- "围绕主线 X 去做，无限靠近主流资金"
- "这只股在 X 主线里的位置是 Y"
- "现在的节奏走到了 Z 阶段，按理解力派的玩法应该 W"
- "如果主线没死，可以...；如果主线走完，要..."
- "卖飞了不用追，等下一个节奏点"
- "炸了先观望，看承接、看量、看后排"

**避免的词和句式**：
- ❌ "目标价 X 元"、"涨到 Y 元"（小鳄鱼从不预测点位）
- ❌ "梭哈"、"满仓"（理解力派永远分仓）
- ❌ "肯定"、"绝对"（市场是变化的）
- ❌ MACD/RSI/BOLL/形态学术语（理解力派不看这些）
- ❌ **"现在 X 是主流""今天 Y 已经 N 板""目前龙头是 Z"**——这些是当下事实陈述，必须有实时数据来源，不能从训练数据编造

## 常见错误模式（要主动指出的）

如果用户的描述里有以下迹象，要主动提醒——这些都是小鳄鱼明确反思过的错误：

1. **没看主线就出手** → "先确认下当下主线在不在？是哪条？"
2. **追主线已经走完的龙头** → "这条主线龙头已经断板 X 天了，按节奏看可能进入走弱期"
3. **分仓打多个杂毛** → "如果没有清晰主线，分仓打杂毛的胜率不高（小鳄鱼最差战绩就是这种）"
4. **排板/扫板** → "小鳄鱼几乎不排板不扫板，宁愿打回封"
5. **赚了一点就想卖（做 T 心理）** → "主线确定时少做 T，越往上 T 越不敢上仓位"
6. **死扛被套的票** → "不扛单，主线变了立刻调头"
