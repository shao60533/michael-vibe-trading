## 案例 1：上海电信弱转强 5 板（华东守住上限的演示）

| 项 | 内容 |
|---|---|
| 标的 | 上海电信 600082 |
| 日期 | 2023-06-09 |
| 性质 | **打弱转强的五板**，当时最高板，换手上去的 |
| 华东原话 | "**这个图形每天看我复盘的都知道，二板的时候我就 YY 它有点妖性，三板转强那天我半路上了，但是四板那天就洗出来了，后面我就没去了，因为我现在不做龙头高标模式，撑死打个二板或二板后的弱转强是我的极限**" |
| 华东操作 | **二板 YY → 三板半路上 → 四板洗出来 → 不再追** |
| 关键纪律 | **"撑死打个二板或二板后的弱转强是我的极限"** |
| **学习要点** | **守住上限是华东的核心纪律**——即使这个票后面继续涨到 5 板，华东在 4 板被洗出后**就不再追**。这是和睿神（敢上重仓敢扛）最大的差别。**对自己能力的清晰认识**：知道自己只能吃二板的肉，不勉强吃 N 板的肉 |

## 案例 3：中码贸易（庄股模板复制 - 反映规律）

| 项 | 内容 |
|---|---|
| 标的 | 中码贸易 001167 |
| 日期 | 2023-06-28 |
| 性质 | **和图三物贸几乎一模一样，主力真是按着同一个模板在坐庄** |
| 华东引用 | "**正如 92 科比所言，很多炒作其实都是在模仿**" |
| 操作 | 弱转强 5 板（同样模板）|
| **学习要点** | **主力的炒作有模板复制**——同一时期不同票走出几乎相同的形态。**识别这种"模板复制"** = 提高出手胜率。**关键**：找到第一个走出模板的票（参考案例），后续票出现同样形态时确认度大幅提高 |

## 案例 4：图解 3 总结（5 案例分析）

| 项 | 内容 |
|---|---|
| 时段 | 2023-06 |
| 5 张图解共性 | "**几乎都是当时辨识度很高的小盘票，虽然都是庄股，但人气很高**" |
| 辨识度定义 | "**什么是人气辨识度，比如要你在人群中找一个人，如果是马云，你看到肯定就认出来了，大家也都容易认出来，如果是个子最高的这个，在人群中鹤立人群也很显然。其他的关注度都太低了**" |
| **学习要点** | **辨识度是华东选股的核心标准**——不是看 K 线漂亮，是看市场关注度。**"人群中找马云"** 是一个极其形象的标准——市场全员都能立刻识别的票 |

## 案例 5：尾盘上 CPO 主线（操作 B 类型）

| 项 | 内容 |
|---|---|
| 主线 | CPO（2023 年算力主线分支） |
| 操作 | **多次在尾盘竞价上主线趋势强票** |
| 入场时机 | 尾盘竞价（14:50-15:00） |
| 优势 | "**情绪盘面更稳定，避免早上追错被 A 杀**" |
| 主线持续性 | 大波段（多次参与） |
| **学习要点** | **尾盘上车的优势**：当日已经能看清盘面强弱 + 避免早上追错。这是华东 30% 操作类型——补足小盘庄股之外的主线机会 |

## 案例 6：2023 全年战绩（整体战绩）

| 项 | 内容 |
|---|---|
| 时段 | 2023 年 10 个月 |
| 收益 | **10 倍**（CAGR 约 1043%） |
| 最大回撤 | **-13.8%** |
| 出手频率 | 1 月 4-5 次 / 2 月 3 只 / 4-5 月合计 10 次 |
| 空仓天数 | **6-7 成** |
| 之前经历 | 7、8 年巨亏（"股 zai 时欠下连本带利上百万的债务"） |
| **学习要点** | **极致的收益 vs 回撤比**——10 倍 vs -13.8% = 比绝大多数游资好。**实现方式**：低频 + 严格上限 + 高胜率 + 果断止损。**从巨亏翻身的故事**说明：风控不是天生的，是亏出来的 |

## 综合案例：华东的操作上限 vs 其他游资对比

| 游资 | 操作上限 | 风格 |
|---|---|---|
| **华东大导弹** | **撑死二板 + 二板后弱转强** | **极致低频低风险** |
| 北京炒家 | 首板隔日卖 | 模式派低频 |
| 采莲路 | 低位首板 | 控回撤派 |
| 92 科比 | 不限 | 全能型 |
| 一瞬流光 | 高位接力 N 板 | 进攻型 |
| 陈小群 | 龙头主升浪锁仓 | 信仰型 |
| 睿神 | 不限 + 跌停翘板 | 极致进攻型 |
| 小鳄鱼 | 不限 | 理解力派 |

**华东的位置**：在所有游资里**操作上限最低**——这是他的纪律，也是他的护城河。

## 怎样使用本案例库（给 Claude 自己）

B 类复盘任务流程：
1. **提取用户操作的核心要素**：是几板？盘子多大？辨识度够吗？是否在主线势内？离场是否果断？
2. **匹配 1-2 个最接近案例**
3. **判断是否符合华东的核心原则**：
   - ✅ 撑死二板（含二板后弱转强）
   - ✅ 小盘（几个亿）
   - ✅ 辨识度像找马云
   - ✅ 借势主线
   - ✅ 次日不及预期立即离场
4. **回 `playbook.md` 找华东会怎么处理**
5. **特别提醒**：
   - 如果用户打了 3 板以上 → 提醒华东上限纪律
   - 如果用户做了大盘票 → 提醒华东只做小盘
   - 如果用户高频出手 → 提醒华东低频是核心
   - 如果用户死扛 → 提醒次日不及预期立即离场

**重要原则**：华东大导弹是**纪律的极致代表**——复盘时**不要看用户这笔操作赚不赚钱**，而是看**是不是符合华东的 5 步出手条件 + 严格上限**。如果不符合，即使赚了也是运气，下次大概率亏回去。
