## 案例 1：见证杯实盘赛（5 个月 23 倍 - 整体战绩）

| 项 | 内容 |
|---|---|
| 时间 | 2021 年初见证杯比赛 |
| 月度收益 | **218%** |
| 5 个月总收益 | **23 倍** |
| 时代背景 | 市场核心抱团崩塌、市场冷清，但**妖债横行 + 创业板活跃 + 新股 5 天无涨跌限制** |
| 主线 | 次新 + 碳中和 |
| **学习要点** | 睿神的爆发主要来自**捕捉时代红利窗口**——新股 5 天无涨跌是估值游戏的黄金机会。**注册制后这种结构性窗口大幅减少**，不要套用 2021 年玩法 |

## 案例 2：顺控大帝（核心共性最佳演示 - 大牛）

| 项 | 内容 |
|---|---|
| 标的 | 顺控大帝（次新 + 碳中和 + 10 厘米） |
| 时间 | 2021-03 |
| 风口位置 | **当时市场最强主流风口（次新 + 碳中和）** |
| 核心共性 | **碳中和 + 10 厘米方向** |
| 结果 | **大牛**（睿神的标志性战绩） |
| **学习要点** | **核心共性的最佳演示**——顺控大帝不仅仅是次新，是"**次新 + 碳中和 + 10 厘米方向**"三重叠加。**任何单一标签都不够**——要找到多重叠加的真核心 |

## 案例 3：西水科技（核心共性不足 - 吃面）

| 项 | 内容 |
|---|---|
| 标的 | 西水科技 688516 |
| 日期 | 2021-03-18 |
| 操作 | **新股上市分时低吸** |
| 睿神判断 | "**有可能是觉得顺控大帝这么牛，作为新股一上来一直回落，不应该，就去低吸了一把**" |
| 结果 | **直接吃面** |
| 失败原因 | "**顺控的牛绝对不仅仅因为是次新，主要还是碳中和这条线，即便是次新，也分为 20 厘米方向和 10 厘米方向。当时市场显然是 10 厘米方向**" |
| **学习要点** | **次新 ≠ 自动牛**——必须看共性。西水是 20 厘米方向，但当时市场炒的是 10 厘米方向 → 方向错就是吃面。**判断联动性，一定要抓住核心共性** |

## 案例 4：寒岭科技（核心共性不足 - 吃面）

| 项 | 内容 |
|---|---|
| 标的 | 寒岭科技 100953 |
| 日期 | 2021-03-19 |
| 操作 | **分时追涨，开盘走人** |
| 睿神判断 | "**依旧是次新，可能还是想做一个顺控大帝的带动次新上涨**" |
| 结果 | 吃面 |
| 失败原因 | "**可惜还是核心共性碳中和概念不足，虽然它带有新能源汽车概念，但是做零部件的和碳中和关联度较差**" |
| **学习要点** | **新能源汽车 ≠ 碳中和**——风口的共性判断要精确到产业链位置。零部件和碳中和关联弱，不能套用碳中和风口的玩法 |

## 案例 5：仁东 4 次失败（高位垃圾庄题材的教训 - 公开反思）

| 项 | 内容 |
|---|---|
| 标的 | 仁东 |
| 操作 | 连续 4 次低吸 / 跌停翘板 |
| 结果 | **连续 4 次失败** |
| 失败原因 | "**误判垃圾庄题材持续度**" |
| **学习要点** | **垃圾庄题材的持续度极低**——即使是顶级游资也会连续 4 次失败。**重仓 + 错判 = 灾难**。教训：①重仓前必须确认风口和共性 ②即使第一次失败也要立刻停手反思，不要连续失败 |

## 案例 6：本期 10 张图解小结（图解 2）

| 项 | 内容 |
|---|---|
| 时段 | 2021-03 |
| 10 张图解结果 | **睿神做的一般，6 次吃面**。中小面为主，肉也是中小肉 |
| **学习要点** | **顶级游资也有低胜率期**——10 次操作 6 次吃面是常态。**关键是单笔损失不大**——中小面而不是大面。这要求**仓位管理 + 严格止损**——睿神虽然敢上重仓，但**重仓 ≠ 不止损** |

## 综合操作类型案例（按 10 种典型操作分类）

### 操作 1-2：次新打板 / 首次放巨量打板
- 见证杯期间高频次新打板
- 关键：放巨量 = 资金真的进来了

### 操作 3：牛股启动反包打板
- 反包 = 多空逆转
- 关键时机：之前的牛股被打回去后

### 操作 4：弱转强打三板（睿神最爱）
- 弱：前一日炸板 / 大阴 / 低开
- 转强：突然放巨量 / 涨停 / 突破前高
- **弱转强 = 资金重新介入 = 最佳入场点**

### 操作 5：分时转强的高位半路
- 信号：高位股盘中分时突然转强
- 关键：要结合题材大小（小题材高位半路风险大）

### 操作 6：连板回调低吸
- 信号：连板票回调 1-2 天
- 关键：强度一般 = 还有回抽空间

### 操作 7：被核后的低吸博地天板
- **新手不要做**
- 翘跌停只在"中位连板首次被核"位置好用
- 多次转弱的不行

### 操作 8：庄股接力
- 信号：庄股连续烂板 → 高开 4 个点直接上车
- 关键：龙头选手要有魄力

### 操作 9：后排跟风补涨
- **两个有效时间点**：
  1. 龙头 5 板打高度后
  2. 龙头高位再起一浪
- 其他时间补涨容易吃面

### 操作 10：跌停低吸 / 跌停翘板（明确不推荐）
- 睿神原话："**不要参与跌停低吸，尤其是新手，尤其是高位跌停**"

## 怎样使用本案例库（给 Claude 自己）

B 类复盘任务流程：
1. **提取用户操作的核心要素**：是否在最强主流风口？核心共性够不够？信号清晰吗？仓位是否符合进攻型？
2. **匹配 1-2 个最接近案例**
3. **判断是否符合睿神的核心原则**：
   - ✅ 站在最强主流风口
   - ✅ 抓住核心共性
   - ✅ 弱转强 / 反包 / 半路等关键信号
   - ✅ 敢上重仓但不梭哈
4. **回 `playbook.md` 找睿神会怎么处理**
5. **特别警告**：
   - 跌停低吸 → 睿神禁区
   - 高位翘跌停 → 只在中位连板首次被核位置
   - 时代红利不可复制 → 2025 年不要套用 2021 年玩法
   - 连续失败 → 像仁东一样立刻停

**重要原则**：睿神是**进攻型代表**，但**进攻不等于鲁莽**。复盘时要看用户**是不是真的站在风口 + 是不是抓住核心共性**。如果不是，即使这笔操作赚了钱，下次大概率亏回去——风口判断错的进攻是地雷。
